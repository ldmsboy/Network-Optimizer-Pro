Guía de uso - Network Optimizer (Web)

  1. Al abrir la aplicación verás un grafo de ejemplo.
  2. Usa 'Hosts' y 'Conexión' para añadir nodos y aristas con peso (latencia ms).
  3. Calcula MST, Dijkstra o la heurística Hamiltoniana desde la sección Algoritmos.
  4. Descarga el grafo en formato JSON o carga uno existente.
  5. La visualización aparece como una imagen PNG; recarga la página para actualizar tras cálculos.
